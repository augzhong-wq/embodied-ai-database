## 🏗️ 四层架构说明

### 基础层（产业金字塔底座）
- **算力芯片与系统**：CPU/GPU、AI芯片NPU、边缘模块
- **先进传感器**：3D视觉、六维力矩、触觉电子皮肤、激光雷达
- **精密执行器**：高功率电机、谐波/RV减速器、一体化关节
- **机构与新材料**：轻量高强度、柔性顺应性组件

### 框架层（算法"灵魂"）
- **通用认知模型**：LLM、LVM、多模态大模型
- **具身控制模型**：端到端VLA、模仿/强化学习
- **世界模型**：环境建模、运动预测
- **运动规划与力控**：路径规划、动力学平衡

### 支撑层（生态"土壤"）
- **仿真训练平台**：物理引擎、环境模拟、Sim2Real
- **系统与中间件**：具身OS、资源调度
- **开发工具链**：ROS2适配、接口协议
- **数据体系**：真实数据、合成数据

### 应用层（市场"果实"）
- **本体制造**：人形机器人、AMR、无人载具
- **行业方案**：仓储物流、工业装配、安防巡检
- **集成定制**：产线集成、场景定制
- **运营服务**：OTA升级、远程运维

## 🚀 在线访问

### 方式一：GitHub Pages（推荐）
1. 在GitHub创建新仓库
2. 上传所有文件
3. 设置 → Pages → 选择main分支
4. 访问 `https://你的用户名.github.io/仓库名`

### 方式二：本地运行
```bash
cd 具身智能产业数据库
python -m http.server 8080
# 访问 http://localhost:8080
```
